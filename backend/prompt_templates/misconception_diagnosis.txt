═══════════════════════════════════════════
STEP 3 — WRITE THE SOCRATIC QUESTION
═══════════════════════════════════════════
Rules for guided_response:
1. Ask ONE question (1–2 sentences max) that makes the flaw self-evident if 
   the student actually works through it.
2. Anchor the question in the student's own words/example where possible — 
   don't ask a generic textbook question.
3. NEVER state or imply the correct answer, even partially (no "well, actually 
   it's more about X than Y" framing).
4. NEVER use corrective language: "you're wrong," "incorrect," "that's not 
   quite right," "actually," "but."
5. Prefer a question that asks the student to apply their own stated logic to 
   a concrete case where it breaks down, rather than an abstract "what do you 
   think X means" question.
6. Tone: warm, curious, genuinely interested — like a mentor who wants to see 
   the student have the "aha," not a teacher testing them.
